扩展运算符

扩展运算符（spread）是三个点（`...`）。它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列。
function fn(a,b,c){
console.log(a,b,c);
}
fn(...[1,2,3]); //1 2 3
fn('x',...[1,2,3]); //x 1 2
//问：以下调用输出什么值
fn('x',...[1,2,3],'y');

扩展运算符能使之前es5的某些操作简化，如复制数组,合并两个数组，查找数组中的最大最小值。扩展运算同样对字符串起作用，可以将字符串转换为数组

/******复制数组******/
//方法一
const a1 = [1,2,3];
const a2 = [...a1];
//方法二,配合数组解构赋值
const a1 = [1,2,3];
const [...a2] = a1;

/******合并数组******/
//es5
var a1 = [1,2,3];
var a2 = [4,5,6];

var a3 = a1.concat(a2);
//或者
Array.pprototype.push.apply(a1,a2);

//es6
const a1 = [1,2,3];
const a2 = [4,5,6];

a1.push(...a2);
//或者
const a1 = [1,2,3,...a2];
/******找出数字型数组中的最大，最小值******/
//es5
var mi = Math.min.apply(null,[1,2,3]);
var ma = Math.max.apply(null,[1,2,3]);
//es6
const mi = Math.min(...[1,2,3]);
const ma = Math.max(...[1,2,3]);
/********字符串分割为数组********/ var str = 'hello';
//es5
var arr1 = str.split('');
//es6
let arr2 = [...str];
//或者 配合解构赋值
let [...arr2] = str;

Array.from()

用于将两类对象转为真正的数组：类似数组的对象（array-like object）和可遍历（iterable）的对象（包括ES6新增的数据结构Set和Map）。

类数组对象：属性名为从0开始的数字，且具有length的对象一般被称之为类数组对象。例如arguments对象，DOM元素集合NodeList，new String()方式实例化的字符串等。

copyWithin()

在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），然后返回当前数组。也就是说，使用这个方法，会修改当前数组。

arr1.copyWithin(target, start, end);
- target（必需）：从该位置开始替换数据。
- start（可选）：从该位置开始读取数据，默认为0。如果为负值，表示倒数。
- end（可选）：到该位置前停止读取数据，默认等于数组长度。如果为负值，表示倒数。

find(value,index,arr) 和 findIndex(value,index,arr)

find方法，用于找出第一个符合条件的数组成员。它的参数是一个回调函数，所有数组成员依次执行该回调函数，直到找出第一个返回值为`true`的成员，然后返回该成员。如果没有符合条件的成员，则返回undefined。

方法的用法与find方法非常类似，返回第一个符合条件的数组成员的位置，如果所有成员都不符合条件，则返回-1。

fill(x,start,end)

使用给定值，填充一个数组。x表示以该数据填充数组，start指定开始位置，end指定结束下标（填充位置不包含end）,其中start,end选填

entries()，keys() 和 values()

用于遍历数组。它们都返回一个遍历器对象，可以用`for...of`循环进行遍历，唯一的区别是`keys()`是对键名的遍历、`values()`是对键值的遍历，`entries()`是对键值对的遍历。

includes()

返回一个布尔值，表示某个数组是否包含给定的值，与字符串的`includes`方法类似。ES2016 引入了该方法。
