ES6 提供了更接近传统语言的写法，引入了 Class（类）这个概念，作为对象的模板。通过`class`关键字，可以定义类。基本上，ES6 的`class`可以看作只是一个语法糖，它的绝大部分功能，ES5 都可以做到，新的`class`写法只是让对象原型的写法更加清晰、更像面向对象编程的语法而已。

//es5
function Person(name,age){
this.name = name;
this.age = age;
}
Person.prototype.sayName = function(){
alert(this.name)
}
var p1 = new Person('李狗蛋',66);
p1.sayName() //'李狗蛋'
//es6
class Person{
constructor(name,age){
this.name = name;
this.age = age;
}
//多个方法之间不加逗号
sayName(){
alert(this.name)
}
}
var p2 = new Person('李狗蛋',66);
p2.sayName() //'李狗蛋'

区别在于es5中，可以在原型（prototype）中添加属性，实例化之后的对象都将拥有该属性（可在__proto__中查看），而在es6的class中只能添加方法，且都是直接添加在原型中的，不能添加属性。

Class 的取值函数（getter）和存值函数（setter）

--取值函数（getter）:设置在获取 定义的属性时，执行的操作
--存值函数（setter）:设置在修改 定义的属性时，执行的操作

class Foo {
get prop(){
return 'hello world';
}
set prop1(value){
console.log(value);
}
}
var f = new Foo();
f.prop = 123;
console.log(f.prop)
